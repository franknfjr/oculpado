# Política de Segurança

## Versões suportadas

Este é um projeto pessoal e apenas a última versão publicada na branch `main`
recebe correções de segurança.

| Versão | Suportada |
| ------ | --------- |
| main   | Sim       |

## Reportando uma vulnerabilidade

Se você encontrar uma vulnerabilidade, **não abra uma issue pública**. Em vez disso:

1. Use o [Private Vulnerability Reporting](https://github.com/franknfjr/oculpado/security/advisories/new)
   do GitHub, ou
2. Entre em contato diretamente com o mantenedor ([@franknfjr](https://github.com/franknfjr)).

Descreva o problema, o impacto potencial e, se possível, um passo a passo para
reproduzir. Você receberá um retorno assim que possível e trabalharemos numa correção
antes de qualquer divulgação pública.

Obrigado por ajudar a manter o projeto seguro.
